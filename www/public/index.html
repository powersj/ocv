Objective

The goal of this project is to create visualizations of online educational courses. This will help educators understand the methods they are using to teach and assess students. The project helps students understand the course and how it is structured. It is specifically targeted for students and teachers using MOOC platforms, like edX.

Check out the demos below!

Video Length

One of the features of the visualizations is the ability to show the length of each video. The visualizer will use the YouTube API to look up the lengths and add them as tooltips when hoveovered with the mouse!

Problem Set Features

A final features is the analysis of components of a course. Each tooltip that shows up for courses will show whether or not platform features have been utilized to deliver value to the student or not.

This is where the project could be extended greatly in the future in an effort to help teachers and students even more with understanding the course content and structure!
