Objective

The goal of this project is to create visualizations of components of entire online education courses to help educators understand the methods they are using to teach and assess students and students to understand how the course is structured. This project is specifically targeted at students and teachers using MOOC platforms, like edX.

Check out the demos below!

Features

Entire Course

See an entire course in one image!

Automated

Connect to MOOCs and have the visualization build!

Interactive

Zoom in and out, move around, expand and hide objects!
